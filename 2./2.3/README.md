# 2.3 통합수준

현행 IFRS 4는 재무회계 목적의 계약 통합수준에 대한 요구사항이 없으며 각국의 현행 회계 관행을 인정함. 반면 IFRS 17은 개별계약은 계약이 지닌 위험과 계약보다 상위의 집합에서 관리되는 범위에 따라 특정 포트폴리오에 귀속되며 수익성과 시간을 고려하여 특정 집합에 포함되도록 규정하는 등 개별계약이 아닌 집합 수준의 인식 및 측정을 요구함.&#x20;
